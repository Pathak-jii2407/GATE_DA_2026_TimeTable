Firestore Rules Helper

Copy these rules into Firebase Console → Firestore Database → Rules, then Publish. They allow each signed-in user to read/write only their own data at users/{uid}/**.

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
